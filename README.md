# 📘 TIL (Today I Learned)

매일 학습한 내용을 기록하는 저장소입니다.  
단순 복사/정리가 아닌 **직접 실습하고 이해한 내용 중심**으로 작성합니다.

## 🔥 Rules
- 하루 1줄 TIL보다 **깊이 있는 문서** 위주로 작성
- 모르는 내용은 "왜?"를 기준으로 정리
- 실습은 스크린샷/예시 코드 포함
- 정답이 아닌 "내가 이해한 방식"을 기록

## 📚 Category
- AWS: EC2, S3, IAM, VPC 실습 중심
- Docker: Dockerfile, Compose, Network, Volume
- Network: VLAN, Routing, NAT, OSPF/EIGRP 등 시뮬레이션 기반
- CS: 운영체제 / 네트워크 핵심 개념
- Git: 협업을 위한 브랜치/머지 전략

## 🚀 목표
- 기술 면접 대비 지식 기반 구축  
- 꾸준한 학습 습관
